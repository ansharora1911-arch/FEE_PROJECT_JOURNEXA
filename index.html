About Journexa

AI Mood Detection

Our AI reads your journal entries and detects your mood using keyword-based sentiment analysis — no external AI API needed.

Happy Things

When you're feeling down, Journexa reminds you of things that make YOU happy — personalized to your preferences.

Auto Mood Chart

The weekly mood chart is updated automatically by AI after every journal entry. No manual mood logging needed.

💾

Local Storage

All data stays in your browser. No server, no signup, no data sharing — 100% private.

Technologies Used

  • HTML5 — Semantic structure (header, nav, main, section, footer)
  • CSS3 — Flexbox, Grid, Media Queries, Gradient Background
  • JavaScript (ES6) — DOM, Events, localStorage, Fetch API
  • Chart.js — Weekly mood chart visualization
  • Sentiment Analysis — AI keyword-based mood detection
  • DummyJSON API — Random motivational quotes
